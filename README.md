# FDK_reconstruction
实现锥束CT正投影和FDK重建，并探讨锥束CT各参数对FDK重建质量影响。

* 基本代码：```fdk_main.py```，其他实验结果参考各文件夹。
* 示例数据：```wugui.mat```，大小为(256, 256, 256)的乌龟数据。
